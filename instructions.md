
# Instructions

This program is a sales analysis application that takes in sales transaction information (in a .csv file), reads and then analyses this information to ultimately generate a report file (report.txt) that allows the business owner to run their business more optimally by making informed financial decisions. The analysis from the report will contain:

- total yearly sales and quantities sold
- breakdown of how much of each product and revenue each product generated per year, date with the amount of sales (and number of items respectively ) occurring a time-interval (i.e. over 6 months, a year, etc.)

In order to use this program, the user must follow these set of instructions:

1. Program requires transaction data that covers these five fields: date, amount, product, count, transaction id, which would be need to be stored in a .csv file
   - must be displayed in this format "date,amount,product,count,transaction_id", where...
       - 'amount' refers to sales generated by product
       - 'count' refers to quantity sold
   - use sales.csv or sample_sales.csv as a reference

2. Users will then be prompted to enter the filename of this accordingly formatted csv file into the terminal and then run the program. This would which would output key information to a csv file called: "report.txt"

Use this command to run the program: `python sales_analyser.py`
